# :recycle: Recycling Helper :recycle:

## 1. 작품 설명

- 실명제를 적용한 쓰레기 불법 투기 근절을 해결하기 위한 실명제 앱입니다.

- 사용자는 회원가입과 로그인을 한 후 바코드를 인식시켜 종량제 봉투를 등록합니다.

- 사용자는 쓰레기통에 종량제 봉투의 바코드를 인식시키고 등록된 봉투이면 쓰레기통이 열려 배출이 가능합니다.
